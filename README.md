# 🌌 Inflation Parameter Visualizer — Web Application

The **Inflation Parameter Visualizer** is an interactive, research-driven web application for exploring and analyzing cosmological inflation models within the slow-roll framework.

---

## 🚀 Flexible Model Input

Users can define an arbitrary inflaton potential:

[
V(\phi)
]

through an intuitive input interface. The system automatically:

* Parses the mathematical expression
* Detects free parameters
* Generates dynamic input controls

➡️ No manual coding required.

---

## ⚙️ Automated Physics Engine

Once a potential is specified, the application performs a full numerical analysis:

* Slow-roll parameters:
  [
  \epsilon(\phi), \quad \eta(\phi)
  ]
* End of inflation condition
* Number of e-folds evolution
* Horizon-exit observables:
  [
  n_s, \quad r
  ]

---

## 📊 CMB-Constrained Predictions

The platform enforces **CMB normalization** automatically, ensuring:

[
A_s \approx 2.1 \times 10^{-9}
]

at the pivot scale, consistent with observational data.

---

## 📈 Interactive Visualizations

Generate high-quality, real-time plots:

* Inflaton potential ( V(\phi) )
* Slow-roll parameters vs. field
* ( n_s - r ) plane with observational bounds
* Scalar and tensor power spectra vs. e-folds
